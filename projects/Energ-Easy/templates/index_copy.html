What does it do?



Some Features


Accurate Energy Predictions

The graph uses hourly weather data from ERCOT and other data sources to produce an accurate model within the industry standard of 3%.


Representations of Historical Data

The model is tested using actual demands from past recordings up to the present. The graph displays comparisons of predictions and actuals for the selected time period.


Interactive Map

The map is drillable, includes overlay capabilities, and allows for the user to see detailed information at any given point in the graph when clicked on.


Information Down to the County Level

The map has intuitive colors and is drillable to a state, region, and county level. Clicking on an area provides summary information about it.
